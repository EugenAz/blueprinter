# Blueprinter
### Configurable CLI. Generate files or simple file structures.

## Prerequisites

CLI requires Node >= 6.9.0, NPM >= 3.

## Table of Contents

* [Installation](#installation)
* [Usage](#usage)

## Installation

**BEFORE YOU INSTALL:** please read the [prerequisites](#prerequisites)
```bash
npm install -g blueprinter
```

## Usage

```bash
bpr --help
```
